## Installation

```bash
brew tap klaas-sh/tap
brew install klaas
```

Or in a single command:

```bash
brew install klaas-sh/tap/klaas
```

## Update

```bash
brew update
brew upgrade klaas
```

## Uninstall

```bash
brew uninstall klaas
brew untap klaas-sh/tap
```

## About

This tap contains the Homebrew formula for [klaas](https://klaas.sh), a remote access wrapper for Claude Code that enables you to control your sessions from anywhere via a web interface.
